Rubirizi Customs Tax Calculator

A specialized digital tool for estimating import duties and taxes for goods entering Uganda, based on **Uganda Revenue Authority (URA)** regulations and the **East African Community (EAC) Common External Tariff**.

## 🚀 Features
* **Automated Duty Calculation:** Covers 0%, 10%, 25%, and 35% EAC CET bands.
* **Tax Breakdown:** Includes VAT (18%), Withholding Tax (6%), and basic levies.
* **Exchange Rate Support:** Converts CIF USD values to UGX for local accuracy.
